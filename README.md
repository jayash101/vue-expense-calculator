# Vue Expense Calculator

A Vue 3 + TypeScript + Vite project for tracking and managing expenses.

## Features

- Add, edit, and delete expenses
- Categorize expenses
- View expense summary and statistics
- Responsive design

## Tech Stack

- **Vue 3** with `<script setup>` SFCs
- **TypeScript** for type safety
- **Vite** for fast development and building

## Project Setup

```bash
npm install
npm run dev
```

For production build:

```bash
npm run build
```
